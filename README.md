# Hotel Reservation System (JDBC)

## Description
A console-based Hotel Reservation System developed using Core Java, JDBC, and MySQL.  
The application performs complete CRUD operations to manage hotel reservations.

## Features
- Reserve a room
- View all reservations
- Get room number by reservation ID
- Update reservation details
- Delete reservations

## Technologies Used
- Java (Core Java)
- JDBC
- MySQL
- IntelliJ IDEA

## Database Details
- Database Name: hotel_db
- Table Name: reservations

## Future Enhancements
- GUI using JavaFX or Swing
- Hibernate ORM integration
